# Random Password Generator

This project is a simple password generator written in Python.  
It creates a random password based on the length entered by the user.

The password includes:
- Uppercase and lowercase letters
- Numbers
- Special characters

---

## How It Works
1. The program asks the user to enter the required password length.
2. It combines letters, digits, and symbols.
3. Random characters are selected to generate the password.
4. The final password is displayed on the screen.

---

## Requirements
- Python 3.x

---

## How to Run
1. Download or clone the repository.
2. Open a terminal in the project directory.
3. Run the program using:

```bash
python Password_Generator.py
